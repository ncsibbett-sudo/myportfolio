Overview

Penguin Surf Dash is an endless runner game inspired by Temple Run, where players control a penguin skiing down icy slopes through procedurally generated obstacles. Built with vanilla JavaScript and HTML5 Canvas, the game features intuitive left/right arrow controls, collision detection, and a scoring system that combines distance traveled with fish collectibles.

The primary purpose of this app is to:

  • Deliver a fast-paced, addictive endless runner with simple controls
  • Demonstrate HTML5 Canvas game development techniques
  • Showcase procedural obstacle generation and collision detection
  • Provide an engaging arctic-themed gaming experience

Product Requirements Document (PRD)

Title: Penguin Surf Dash Version: 1.0 Date: September 12, 2025

1. Overview

Summary

Penguin Surf Dash is an endless runner game inspired by Temple Run, where the player controls a penguin riding a surfboard through an icy obstacle course. The player uses arrow keys to make quick decisions—jumping, sliding, and dodging obstacles—while racing to achieve the highest score possible.

Goals

  • Deliver a fast-paced, addictive endless runner
  • Keep controls intuitive and responsive
  • Use a unique penguin-and-surfboard theme to differentiate from similar games

2. Core Gameplay

Controls

  • Left Arrow → Move left lane
  • Right Arrow → Move right lane
  • Space → Start game

Game Loop

  • Player starts at the top of an icy slope with the penguin already in motion
  • Obstacles and pathways are procedurally generated to increase replayability
  • The penguin accelerates gradually, making reaction times shorter as the run continues
  • The run ends if the penguin crashes into an obstacle

Scoring System

  • Distance traveled: Primary score metric
  • Collectibles: Fish tokens scattered on the path give bonus points
  • Combined Score: Distance + fish collection for final score

3. Technical Implementation

Current Features

  • Three-lane movement system
  • Procedural obstacle generation
  • Progressive speed increase
  • Collision detection
  • Score and distance tracking
  • Game over and restart functionality

Future Enhancements

  • Power-ups (speed boost, invincibility shield, magnet for fish)
  • Daily challenges/quests
  • Character customization (different surfboard designs, outfits)
  • Multiplayer leaderboards
